# ブラウザ版ねいねいチャットbot

`index.html`をブラウザで開くと、LINE風のチャット画面で「ねいねい」botを試せます。

画面右側で次を変更できます。

- 名前
- 相手の名前（`〇〇`に入る名前）
- 肩書き
- 性格
- 口調
- 決め台詞
- 会話スタイル
- 返答の長さ
- 絵文字の有無

性格、肩書き、決め台詞に`〇〇`と書くと、画面で入力した相手の名前に置き換わります。

## 雑談対応

ブラウザ版では、短い相づち、好きなもの、相談、迷い、質問、近況、眠さ、食べもの、天気っぽい話などをルールベースで受け取ります。

本当に「なんでも自然に会話できる」状態を目指す場合は、LINE移行時にOpenAIなどのAI APIへ接続し、ここで作ったキャラクター設定をプロンプトとして渡す構成にします。

## ねいねい設定

初期設定は「ねいねい」にしています。ブラウザに以前の設定が残っている場合は、画面右側の「ねいねい設定に戻す」を押すと、ねいねいの名前・一人称・口調・決め台詞に戻ります。

## LINEへの移行

このデモのキャラクター設定と返答処理は`script.js`にあります。LINE版に移すときは、`character`設定と`createReply()`をサーバー側の共通会話エンジンへ入れて、LINE Webhookから呼び出します。
